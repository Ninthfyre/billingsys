# Pool Cafe Manager

A modern, responsive single-page pool table management dashboard built with HTML, CSS, and Vanilla JavaScript.

## Features
- Manage multiple tables
- Start/end game sessions
- Live timers and billing
- Searchable game history
- LocalStorage persistence
- Settings and backup export/import

## Run locally

```bash
python -m http.server 8000
```

Then open `http://localhost:8000/` in the browser.
